# Reference Directory

이 디렉토리는 선행연구 검토와 novelty framing을 위한 별도 보관소다.

## Contents

- `reference_matrix.md`
  source-grounded literature review 메모
- `reference_matrix.csv`
  정렬과 필터링이 쉬운 tabular matrix
- `references.bib`
  검증된 citation 모음
- `sources/`
  로컬에 내려받은 원문, landing page, 공식 repo README, BibTeX

## Usage

- RQ와 novelty 문장은 이 디렉토리를 먼저 통과한 뒤 확정한다.
- 새 reference를 추가할 때는 `sources/` 로컬 사본부터 확보한다.
